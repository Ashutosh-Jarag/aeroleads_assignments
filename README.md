# AeroLeads Projects

This repository contains a collection of projects developed for AeroLeads, showcasing a range of capabilities from web scraping and automation to AI-powered content generation.

## üöÄ Projects

Here's a brief overview of the projects included in this repository:

### 1. [AI AutoDialer](./autodialer-app/)

A web-based application that automates sales calls. It uses Google's Gemini AI to generate personalized scripts and Twilio to place the calls, with a simple Flask interface to manage the process.

-   **Technologies**: Python, Flask, Twilio, Google Gemini AI
-   **Go to project**: [`./autodialer-app/`](./autodialer-app/)

### 2. [AI Blog Generator](./blog_generator/)

An AI-powered tool that automatically writes complete blog posts from a list of titles. It features a web interface and leverages LangChain and LangGraph for a structured content creation workflow.

-   **Technologies**: Python, Flask, LangChain, LangGraph, Google Gemini AI
-   **Go to project**: [`./blog_generator/`](./blog_generator/)

### 3. [LinkedIn Job Scraper](./job-scraper/)

A web scraper that automatically collects job postings from LinkedIn. It's built with Selenium and BeautifulSoup and is designed to be easily customizable for different job roles.

-   **Technologies**: Python, Selenium, BeautifulSoup
-   **Go to project**: [`./job-scraper/`](./job-scraper/)

### 4. [LinkedIn Profile Scraper](./linkedln-scraper/)

A sophisticated scraper for gathering data from LinkedIn profiles. It includes features for automated login and employs several anti-detection measures to ensure reliable performance.

-   **Technologies**: Python, Selenium, `webdriver-manager`
-   **Go to project**: [`./linkedln-scraper/`](./linkedln-scraper/)
